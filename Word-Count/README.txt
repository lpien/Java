In ArrayTree.java the class implements a data array of type Comparable.  This class includes several methods that are attributes of the data array such as the size, height, depth, insert, find, and a method that prints the array (or nodes) in order.  Word.java is a class for a word that is found - it is a place to store characteristics of the strings.  For example each
Word() has a string and a count.  The printConsumer class has a method that counts the number
of words.
